Docker 中搭建 Neo4j 图数据库

2020-02-15 04:43:43 1 min read # docker # 图数据库

1. 在 docker hub 下查看 neo4j 镜像

docker search neo4j

2. 拉取 neo4j 镜像

docker pull neo4j

拉取完成后输入命令可以看到镜像已经存在

docker images

3. 启动 neo4j 镜像

执行命令

docker run \
    --publish=7474:7474 --publish=7687:7687 \
    --volume=$HOME/neo4j/data:/data \
    --rm -d neo4j

​ 参数说明：

  • publish: 配置本机与容器之间的映射端口,

  • volume:配置共享的宗卷，用于数据持久化

4. 启动 neo4j 自带的前端页面

在浏览器中输入

http://localhost:7474/browser

可以看到,这里的端口是7474,是我们 run 命令里面设置的.还有一个端口7687用来连接数据库

5. 连接neo4j

账号密码默认都是 neo4j

至此就连接上了neo4j
